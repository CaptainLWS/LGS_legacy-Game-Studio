Legacy Engine — Interactive Docs

Use the embedded interactive example to experiment with the engine. The example includes buttons to auto-load modules, spawn NPCs, and pause the simulation.

How it works

  • The engine is a tiny ECS-style runtime (engine/engine.js).
  • Modules live in engine/modules/* and export an init(engine, options) function.
  • The module loader (engine/module_loader.js) can auto-load modules based on engine/modules/modules.json.
