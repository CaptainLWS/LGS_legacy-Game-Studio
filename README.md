
## Engine & Catalog

This repository acts as the "command bridge" for Legacy Game Studios. It hosts the studio website and a lightweight engine reference implementation. The engine itself and related catalogs live across a set of repositories and packages:

- Legacy Engine Core: engine/ (core engine materials and specs in this repo)
- Demo Game Catalog: https://github.com/CaptainLWS/LGS-Demo-Game-Catalog-Repo
- Kid-Safe Games: https://github.com/CaptainLWS/KSG_Kid-Safe-Games

Think of this repo as the command bridge and the other repos as the engine decks — together they form the studio fleet.
