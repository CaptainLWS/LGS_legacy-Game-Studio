# Legacy Game Studios Website

![Legacy Game Studios](https://raw.githubusercontent.com/CaptainLWS/LGS_legacy-Game-Studio/main/assets/logo.svg)

## 🎮 About

Legacy Game Studios is a premium game development and digital media production company dedicated to creating highly detailed, cinematic games and productions. Our mission is to provide games, movies, and entertainment that stimulate cognitive development for audiences of all ages.

## ✨ Features

- **Responsive Design**: Fully optimized for desktop, tablet, and mobile
- **Premium Styling**: Gold and black theme with smooth animations
- **Game Showcase**: Display your portfolio of games
- **Productions Section**: Highlight your digital films and series
- **Contact Form**: Easy communication with visitors
- **Social Media Integration**: Links to all major platforms
- **Fast Performance**: Optimized for quick loading

## 🚀 Quick Start

### View Live Site
https://CaptainLWS.github.io/LGS_legacy-Game-Studio/

### Local Development

1. Clone the repository
```bash
git clone https://github.com/CaptainLWS/LGS_legacy-Game-Studio.git
cd LGS_legacy-Game-Studio
```

2. Open `index.html` in your browser or use a local server
```bash
python -m http.server 8000
# Visit http://localhost:8000
```

## 🎨 Customization

### Update Content
Edit `index.html` to:
- Change game titles and descriptions
- Update production information
- Modify about section
- Update contact details

### Update Styling
Edit `styles.css` to:
- Change colors (primary: `#ffd700` gold)
- Adjust layout
- Modify animations
- Update responsive breakpoints

### Add Functionality
Edit `script.js` to:
- Add form submission
- Create game filters
- Add analytics
- Implement features
